\vspace{-0.35cm}

\paragraph{Other bitwise operators}\

All other bitwise operators take two \St{uint32} values obtained from the specified operands (see \hyperlink{types:twos_complement}{the respective section} for details) and produce a \St{uint32} value as a result. The right hand side operand must be less than the size of a machine word (i.e.\ no more than 31), otherwise an execution error occurs.
