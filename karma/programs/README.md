# Programs

This directory provides programs written on the Karma assembler language.

That includes:

* Sample programs from the [docs](../docs/Karma.pdf)

* A number of [playgrounds](playgrounds), some of which have mainly the same
  functionality as the sample programs from the [docs](../docs/Karma.pdf),
  while others may implement different tasks

* A [printing library](print) fully implemented on the Karma assembler language
